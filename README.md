# AI Data Intelligence Pipeline

Data Intelligence pipeline for large-scale ingestion of:

- Research Papers
- AI Jobs
- AI News
- Startup Data
- Product Data

## Features

- Entity Resolution
- LLM Fallback Routing
- Retry Logic (429 Handling)
- CSV Export
- JSON Export
- GitHub Metrics Extraction

## Current Data Sources

- Arxiv API
- TechCrunch AI Feed
- WeWorkRemotely RSS

## Outputs

- research_papers.csv
- research_papers.json
- jobs.csv
- news.csv
